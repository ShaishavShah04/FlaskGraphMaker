Instructions

This is a simple graph making tool created for personal use. It can make simple graphs with basic options such as linear regression and error points.

Input Fields

Under the 3 text input tabs, input the desired corresponding labels.

Axis Integers

This option is primarily only used if your data set involves tiny numbers. If your want axis's numbers to have a potential increments of between 0 - 1, click the option "Integers allowed". This will set your axis's open to having numbers in the format x.yz. If you prefer axis numbers to be only integers, click "No Integers".

Graph Options

This options are fairly straight forward. You can choose between linear regression, a line connecting the data points, and a simple scatter plot

Notice: For the line connecting option, the line will connect the points from top to bottom as put on the uploaded .csv file. The line will NOT take into consideration the x-values.
